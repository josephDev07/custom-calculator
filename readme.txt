
// Custom Calculator
Custom Calculator est une application personnalisée conçue pour effectuer des opérations 
mathématiques de base avec une approche intuitive et conviviale.

// Objectif du projet :
Créer une calculatrice sur mesure, facile à utiliser et adaptée à divers scénarios, 
qu’ils soient éducatifs, professionnels ou personnels.

// Fonctionnalités principales :
- Addition : Effectuez des additions rapides et précises.
- Soustraction : Calculez la différence entre deux nombres sans effort.
- Division : Divisez deux nombres avec une gestion des erreurs, comme la division par zéro.
- Interface simple : Une expérience utilisateur claire et directe pour tous les niveaux.
- Extensible : Ajoutez ou modifiez facilement des fonctionnalités selon vos besoins.
